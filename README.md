# Pixel Rooms

Desktop application for Windows. USB camera recording + video editing with ffmpeg.

## Stack
Electron · Next.js 15 · Mantine 7 · TypeScript · React 18

## Quick start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output: `dist-app/Pixel Rooms Setup 1.0.0.exe`
